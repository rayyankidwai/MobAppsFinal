## Application Overview
FoodieScore is a mobile application that allows users to rate, review, and track food items from different restaurants. The app is designed to help users remember what they’ve tried, find their favorite meals, and make better food choices in the future.

---

## Project Goals
- Provide an easy way to log and rate food experiences  
- Help users track favorite meals and restaurants  
- Build a simple, clean, and interactive mobile app  
- Demonstrate understanding of mobile app development concepts  

---

## Key Features
- Add food items with:
  - Food name  
  - Restaurant name  
  - Rating (1–5 stars)  
  - Optional review/notes  
  - View all rated foods in a list  
  - Edit or delete food entries  
  - Sort foods by rating  
  - Highlight top-rated dishes  
  - Simple and user-friendly interface  

---

## Application Architecture

The app will follow a **component-based architecture using React Native**.

### Main Components
- **App.js**  
  Controls overall application flow and state  

- **Food List Screen**  
  Displays all saved food entries  

- **Add Food Screen**  
  Allows users to input new food ratings  

- **Food Card Component**  
  Displays individual food details (name, rating, review)  

- **State Management**  
  Uses React state (useState) to store and update food data  

- **Stylesheet**  
  Handles layout, colors, and UI styling  

---

## Data Flow
1. User enters food details  
2. Data is stored in application state  
3. Food list updates automatically  
4. User actions (add/edit/delete) update the UI in real-time
